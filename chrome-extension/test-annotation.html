Enable the extension by pressing the icon

How to use:

  1. Click "select new tag".
  2. Move your mouse over elements to highlight them.
  3. Right-click the element you want to annotate.
  4. This can be enabled on any website and it will work just the same.

The idea behind this is that generally relevant content will be confined to a tag

Documents don't have to be pre-compiled by wrapping every word into tags, which might break the page styles
